# Agenda MotorByte 📅🚗

Proyecto desarrollado en Java con Spring Boot para la gestión de agendas y coordinación de vehículos y vendedores, como parte del ramo de Desarrollo Full Stack.

## Descripción

Este microservicio permite administrar agendas relacionadas con vehículos y vendedores, incluyendo:

- Registro de agendas
- Consulta de agendas
- Asociación de vehículos y vendedores
- Persistencia en base de datos
- Migraciones automáticas con Flyway

---

# Tecnologías utilizadas

- Java
- Spring Boot
- Maven
- MySQL
- Spring Data JPA
- Flyway
- REST API
